🚀 Etsy Upload Guide — Your 15 Products

!

Before You Start

Open this folder on your computer — all your product files are here:
C:\Users\prisc\Documents\Athena-Public\etsy_products\wall_art\

Keep this guide open in one tab, Etsy in another. You'll copy-paste from the listing files in etsy_products\listings\

1

Go to Your Shop Manager

Click your profile icon (top right) → Shop Manager → Listings → Add a listing

2

Set Listing Type

  • Type: Digital (not Physical)
  • "Who made it?": I did
  • "What is it?": A finished product
  • "When was it made?": 2020-present
3

Upload the Product Image

Under "Photos" — drag the PNG file from your wall_art folder. This is both the listing image AND the download file.

Under "Digital files" — upload the SAME PNG file. This is what the buyer downloads after purchase.

4

Copy-Paste the Title

Open the listing file (batch_1_listings.md or batch_2_listings.md) and copy the Title for that product. Paste into Etsy's title field.

Titles are SEO-optimised — use them exactly as written.

5

Set Category

Search for and select:

Art & Collectibles > Prints > Digital Prints
6

Copy-Paste the Description

Copy the full Description from the listing file and paste it into Etsy's description box. The emojis and formatting will transfer.

7

Add Tags (Critical for SEO!)

Copy each tag from the listing file's Tags line. Add them one at a time into Etsy's tag field (press Enter after each). Use all 13 tags.

8

Set Price

  • Elegant prints: NZ$5.99
  • Trippy + Trump prints: NZ$6.99
9

Publish

Scroll down → click "Publish". Done. That's one listing live. Repeat for the next product.
