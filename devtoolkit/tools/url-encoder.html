About URL Encoding

URL encoding (percent-encoding) converts special characters into a format that can be safely transmitted in URLs. Characters like spaces, &, =, and ? have special meaning in URLs and must be encoded when used as data. encodeURIComponent encodes all special characters (for query parameter values), while encodeURI preserves URL structure characters like : / ? # @ &.
