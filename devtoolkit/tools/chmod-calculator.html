About File Permissions

Unix/Linux file permissions control who can read, write, and execute files. Each permission type has a numeric value: Read (4), Write (2), Execute (1). These are summed for each class: Owner, Group, and Others. Common patterns: 755 (owner full, others read+execute), 644 (owner read+write, others read-only), 600 (owner read+write only).
