# Athena Knowledge Index

> **Purpose**: This file maps the categorical structure of the AI's memory matrix. 
> Whenever an AI needs to retrieve specific information, refer to this map to understand which directory to target for search tools. DO NOT scan the entire directory tree unless a search within the target directory fails.

## `health/`
Contains all biological, fitness, and medical data.
- **DNA Data:** `health/DNA_Analysis_Report.md`
- **Blood Panels:** `health/blood_tests/`, `health/ultrathink_blood_analysis.md`
- **Supplements:** `health/ultrathink_supplement_analysis.md`, `health/supplement_sourcing_nz.md`
- **Fitness Protocol:** `health/home_workout_protocol.md`

## `finances/`
Contains all budgetary, e-commerce, and procurement data.
- **Shopping Cart / Future Purchases:** `finances/purchase_queue.md`
- **Product Sourcing Research:** `finances/water_flosser_sourcing.md`

## `core/`
Contains the user's operational philosophies, personal identity, and system directives.
- **Top Level Directives:** `core/agency_directives.md`, `core/life_engine_directive.md`
- **Daily Manifesto:** `core/daily_manifesto.md`
- **User Parameters:** `core/user_profile.md`

## `logs/`
Contains chronological, historical daily logs and weekly reflections.
- **Daily Logs:** `logs/daily_log_YYYY-MM-DD.md`
- **Reflections:** `logs/reflection_YYYY-MM-DD.md`

## `maintenance/`
Contains standard operating procedures (SOPs), queues, and routines outside of direct health/bio.
- **Routine Management:** `maintenance/routine_queue.md`
- **Pet Care:** `maintenance/dog_training_protocol.md`
