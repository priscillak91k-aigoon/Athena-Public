# F-P-P Protocol (Feelers–Probes–Payloads)

## Date Added: 10 December 2025

## 2.1 Core Problem: The Doom Loop

```
Damaged Schema (L4/5): "I'm inherently not good/interesting enough; 
                        must use massive effort to prove value."
    ↓
Damaged Behaviour (L2): Send 3-page deep analysis to low-interest target 
                        (high-cost Payload immediately)
    ↓
Bad Outcome (L1): Target spooked or uninterested → silent (soft rejection)
    ↓
Re-traumatisation: Interprets silence as "I poured out my soul and was 
                   still rejected → I'm truly worthless"
```

## 2.2 Three-Tier Investment Protocol

| Tier | Cost | Definition | Example |
|------|------|------------|---------|
| **Feeler** | 1 unit | Ultra low-cost probe | A question, a link |
| **Probe** | 10 units | ONLY if Feeler gets positive response | A paragraph, deeper question |
| **Payload** | 100 units | ONLY if Probe gets reciprocal investment | The 2-3 page deep content |

## 2.3 Feeler Examples

```
Low-investment probes:
├─ "Saw an article about [shared interest], thought of you — what do you think of X?"
├─ "How have you been?"
├─ [Share a relevant meme]
└─ "Free this week?"
```

## 2.4 Escalation Conditions (Must Follow Strictly)

```
Feeler → Positive response?
  ├─ No → STOP. Don't send Probe.
  └─ Yes → Send Probe

Probe → Reciprocal investment?
  ├─ No → STOP. Don't send Payload.
  └─ Yes → Send Payload
```

## 2.5 Defining "Positive Response"

| Response Type | Positive? | Next Step |
|---------------|-----------|-----------|
| Substantive reply + counter-question | ✅ Yes | Upgrade to Probe |
| Substantive reply, no counter-question | ⚠️ Neutral | Send another Feeler to observe |
| "Haha" / "OK" / "👍" | ❌ No | Stop investing |
| Read but no reply | ❌ No | Stop investing |
| Reply hours/days later | ❌ No | Deprioritise |

## 2.6 Outcome Comparison

| Old Mode | New Mode |
|----------|----------|
| Send Payload directly → Target didn't even pass Feeler test | Protocol auto-terminates at step 1 |
| Massive emotional and energy loss | Loss ≈ 0 |
| Re-traumatisation event occurs | Re-traumatisation event never happens |
