# Protocol: LinkedIn Publication Strategy (Athena)

> **Purpose**: Standard Operating Procedure (SOP) for high-impact LinkedIn distribution.
> **Source**: Internal case study — "SkillsFuture Hack"

## 1. Format Selection: The Trap

* **Article (Pulse)**: creating a blog post. Good for SEO/Permanence. **BAD for Feed Reach.**
* **Post (Update)**: Standard status update. **GOOD for Virality (5x-10x views).**
* **SOP**: Always default to **Post**.
  * If content is long (>1500 chars), publish as **Article**, BUT immediately create a **Teaser Post** to link it. Never rely on Article distribution alone.

## 2. Visual Strategy

* **Pattern**: "Urgency vs Benefit" Split Screen.
  * *Left*: The Pain/Deadline (e.g., Red Calendar "31 Dec").
  * *Right*: The Reward/Unlock (e.g., Golden Ticket, Access Card).
* **Prompting**: Use `Imagen 3` or similar. Focus on photorealistic, high contrast, text overlay.
* **Metadata**: Always add Alt Text and "Image Credit" (e.g., "Concept by [Your Brand]") to protect IP/brand.

## 3. Copywriting Angles

* **The "Hacker" Angle** (High Engagement): Focus on ROI, "beating the system," and math. "Don't burn money."
* **The "PSA" Angle** (Safe): Helpful alert. "Just a reminder."
* **Signature**: Use `*Co-authored with [your AI model of choice].*` to signal AI-Native branding.

## 4. Distribution Checklist

1. [ ] **Headline**: Does it stop the scroll? (e.g., "7 Days Left").
2. [ ] **Math**: Is the ROI clear? ("1 day vs 1 year").
3. [ ] **Call to Action**: Direct link + deadline warning.
4. [ ] **Footer**: Hashtags + Co-author signature.
5. [ ] **First Comment**: If linking external heavy content, consider putting link in comments (optional, debated algorithmically).
