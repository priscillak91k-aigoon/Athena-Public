# Protocol SG-002: Face Culture Physics

> **Source**: Internal heuristic (informed by cross-cultural psychology research on collectivism, face culture, and perspective-taking)  
> **Domain**: Singapore-Specific / Social Physics  
> **Priority**: ⭐⭐ High  
> **Note**: Specific percentage weights (70-85%) are approximations based on observed patterns, not direct citations.

---

## Core Principle

> **In Singapore (and broader East Asian contexts), third-party observer perception (L3) has 70-85% weight—far more than Western contexts.**

Face culture amplifies all social consequences by 1.5-2.5×.

---

## Three-Layer Image System

| Layer | Weight | Function | Detection |
|-------|--------|----------|-----------|
| **L1: Internal** | 5-10% | What you know/feel/intend | "What am I thinking?" |
| **L2: Participant** | 15-25% | Direct participant's experience | "How do they see me?" |
| **L3: Observer** | 70-85% ⭐ | Third-party inference from observable | "What would a stranger think?" |

**⚠️ L3 has ABSOLUTE VETO POWER**

## Avalanche Mechanism

```
Resentment Accumulation (Silent):
├─ Small withdrawals accumulate
├─ Target doesn't confront (face preservation)
├─ Resentment builds in private
└─ Appears "normal" on surface

Permission Event (Trigger):
├─ You make observable mistake
├─ External authority validates criticism
├─ Group dynamic shifts against you
└─ "Permission" granted to Express resentment

Activation = Avalanche:
├─ All accumulated resentment releases at once
├─ Others join (bandwagon effect)
├─ Narrative solidifies against you
└─ Recovery extremely difficult ☠️
```

## High-Risk Scenarios (SG-Specific)

### Scenario 1: CNY Gathering (Extended Family)

```
L3 Weight: 85%
Stakes: Generational (parents' face affected)
High-Risk Actions:
├─ Discussing salary directly
├─ Comparing with cousins
├─ Criticizing parents' decisions
├─ Showing off achievements
└─ Political opinions

Safe Strategy:
├─ Deflect personal questions
├─ Redirect to asker
├─ Understate achievements
├─ Praise elders
└─ Keep opinions generic
```

```
L3 Weight: 90%
Stakes: Career-length reputation
High-Risk Actions:
├─ Correcting boss publicly
├─ Taking credit individually
├─ Showing frustration
├─ Disagreeing directly
└─ Complex explanations (shows others up)

Safe Strategy:
├─ Private feedback only
├─ Credit the team
├─ Emotional neutrality
├─ "Building on that idea..."
└─ Match group's technical level
```

### Scenario 3: NS Bunking

```
L3 Weight: 95%
Stakes: Permanent (20-30 witnesses, 2-year loop)
High-Risk Actions:
├─ Showing weakness
├─ Complaining excessively
├─ Superiority display
├─ Deep personal disclosure
└─ ANY emotional outburst

Safe Strategy:
├─ L1-L2 information only
├─ Blend with median
├─ Silent competence
├─ Help but don't show off
└─ Never assume privacy
```
