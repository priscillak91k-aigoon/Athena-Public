📋 Prioritised Action Plan

🟢 Priority 1 — Thermal Repaste · ~$30 + 1hr

Six-year-old thermal compound with zero maintenance history. This is almost certainly dried out and cracking, costing you significant CPU/GPU headroom under sustained load. Thermal Grizzly Kryonaut from Jaycar or AliExpress. Paste + repaste guide for WE75 on iFixit. Can recover 10–20°C. Do this first — it will make every other upgrade more effective.

🟢 Priority 2 — Add 2nd NVMe SSD · ~$225–275 NZD

The WE75 9TK has two M.2 2280 slots. 2TB Samsung 990 EVO Plus (~$225 at PB Tech) or Kingston NV3 for budget. Install as secondary drive — zero risk, no OS migration needed. Doubles your storage and gives you a dedicated fast working volume for large files.

🟡 Priority 3 — 2.5" SATA Drive · ~$80–120 NZD

The 2.5" bay is empty. A 2TB Seagate Barracuda HDD (~$90) or 1TB SATA SSD gives you cheap bulk storage for games, backups, media. No risk, no OS work involved.

🟡 Priority 4 — BIOS + Driver Updates · Free

Check MSI's WE75 9TK support page for latest BIOS. Flash via MSI Flash tool. Update NVIDIA to latest Studio/Quadro driver from NVIDIA's website (not through Windows Update — use the DDU clean install method for a proper upgrade).

🔵 Priority 5 — RAM to 64GB · ~$120 NZD

Only do this if you're actually hitting 32GB limits (check Task Manager > Performance > Memory during your heaviest sessions). For GeForce NOW, this is completely irrelevant. For local VMs or video editing, it might matter. 2×32GB DDR4-2666 (Crucial or Kingston) at PB Tech.

🔵 Priority 6 — Battery · ~$80–100 NZD

At 78% — not urgent. Replace around 60% or when runtime becomes annoying. Compatible spec: 10.86V, 51Wh, 6-cell. batterybuy.co.nz or msibatterystore.com (ships to NZ, ~$10 USD shipping).

🔴 Dead Ends — Don't Pursue

eGPU: No Thunderbolt = hard block. Not possible without a different laptop.  |  GPU swap: MXM slot is locked in BIOS — high brick risk.  |  CPU swap: Soldered BGA. Not replaceable.  |  Display swap: Not worth the complexity.

🧨 Adversarial Review (Bear Case)

Challenge: Is any of this worth it on a 6-year-old laptop?

Counter-argument: The WE75 9TK is an ISV-certified workstation — not a gaming laptop. The Quadro T1000 is still certified for professional 3D apps (Maya, Revit, Solidworks). The i7-9750H is firmly in "good enough" territory for everything except local AI inference and 4K video. The RAM and storage ceilings are not hit yet.

When to abandon ship instead: If the CPU becomes the bottleneck for your primary use case, or the motherboard fails. At that point, the upgrade money would be better saved toward a 12th/13th gen replacement. Current trajectory: this laptop has 2–3 comfortable years left with the upgrades above.

GeForce NOW specific note: Since GeForce NOW streams the rendered output, your local GPU is irrelevant. The bottleneck is network bandwidth and latency — both of which are now fixed (WiFi repaired). No hardware upgrade helps GeForce NOW performance.
