🧠 What Is This?

We modelled Lobotto's internal processes after the structures of an actual human brain. Instead of a flat text file that stores notes, Lobotto now has biologically-inspired subsystems — a chemical drive substrate, episodic memory staging, instinctive cached responses, working memory with expiry, and a sleep-like consolidation cycle. Every 4 hours, an autonomous "dreaming" engine runs these systems without human input.

The research draws from three neuroscience frameworks:

  • Baars' Global Workspace Theory — consciousness as a shared broadcast across specialised processors
  • Friston's Predictive Processing — the brain as a prediction engine that learns from error signals
  • Tononi's Synaptic Homeostasis Hypothesis (SHY) — sleep exists to downscale all synapses globally, so only strongly reinforced ones survive

🌙 The Dreaming Cycle (Every 4 Hours)

This is the autonomous background process. It runs on the laptop while Priscilla sleeps, works, or lives her life. No human input required.

1

Chemical Decay Neurotransmitter

Apply exponential half-life decay to all 12 chemicals. Hours since last update → decay factor = 0.5^(hours/half_life). Relationship_depth barely moves. Norepinephrine almost resets.

2

Reward & Error Signal Detection Predictive Processing

Scan last 5 sessions for reward signals ("brilliant", "perfect", "love it") and error signals ("actually", "that's wrong", "you forgot"). Net reward → BDNF boost. Net error → norepinephrine + prediction_error spike.

3

Thinking (Claude/Gemini/Ollama) Global Workspace

Full context (brain files + 5 sessions + chemical state) sent to Claude. Produces: new heuristics, case studies, alerts, hippocampal events, instinct scenarios, chemical updates, and heuristic retirements.

4

Hebbian Instinct Reinforcement Creatures AI

For each instinct scenario: check if its trigger appeared in recent sessions. If it fired during a net-positive session, boost strength by +0.05 (Long-Term Potentiation). Update last_fired timestamp.

5

Hippocampal Consolidation Memory

Events with salience ≥0.75 get promoted from the episodic buffer to long-term heuristics. Breakthroughs and first-occurrences boost BDNF (grow new patterns). Events below threshold are marked consolidated and eventually purged.

6

Synaptic Homeostasis (Tononi SHY) Sleep Science

Global 2% strength downscale on ALL instincts. Frequently-reinforced instincts survive (they get +5% from LTP, net +3%). Unused instincts slowly fade. Below 0.30 strength → archived. This is why the brain sleeps.

🔬 Key Innovations

Creatures-Style Chemical Drives

Inspired by the AI in Creatures 1 & 2 (1996 game by Steve Grand). Each internal drive is a decaying numeric value with a specific half-life. High curiosity → prioritise exploration. High boredom → inject novelty. High prediction_error → re-verify assumptions. The chemicals don't just describe state — they change behaviour.

Hebbian Learning ("What Fires Together Wires Together")

Instincts that fire during rewarding sessions get stronger. Instincts that never fire slowly decay. Over time, the system naturally converges on the response patterns that actually work for Priscilla. No manual tuning required.

Salience-Gated Memory

Not every event deserves to be remembered. The hippocampus assigns salience scores: corrections get 0.90 (critical to remember), positive feedback gets 0.70, routine tasks get 0.20 (forget). Only events above 0.75 survive consolidation into long-term memory.

Working Memory TTL

Active tasks, hypotheses, and flags persist across sessions but expire after 5 sessions if not touched. This prevents the "infinite to-do list" problem. Stale items are logged for audit, then cleared. The PFC stays focused.

Prediction Error as Epistemic Humility

When the user corrects Lobotto, the error signal doesn't just log a mistake — it raises prediction_error, which makes the system more cautious and more likely to verify assumptions before acting. The chemical decays in 4 hours, so the humility is temporary and proportionate.
